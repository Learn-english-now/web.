Spelling Changes with Suffixes

Learn the rules for doubling consonants and changing 'y' to 'i' when adding suffixes.

Introduction

Adding suffixes like -ing, -ed, -er, or -ly can change the spelling of the original word. Understanding these rules helps prevent common mistakes and improves writing accuracy. This section will guide you through the most important spelling changes when adding suffixes in English.

Doubling Consonants

When a single-syllable word ends with a consonant preceded by a single vowel, the final consonant is usually doubled before adding a suffix that begins with a vowel.

Example: run → running, sit → sitting, begin → beginning.

However, if the word has two syllables and the stress is on the first syllable, we usually do not double the consonant (e.g., "visit" → "visited").

Changing 'y' to 'i'

When a word ends in a consonant + "y", the "y" changes to "i" before adding suffixes like -ed, -er, or -es.

Note: The "y" does not change to "i" when the suffix starts with "i", like in "playing".

Dropping the Final 'e'

If a word ends with a silent "e", we usually drop the "e" before adding a vowel-starting suffix like -ing or -able.

  • Make → Making
  • Use → Usable
  • Ride → Riding

Exception: Keep the "e" if removing it would cause confusion, like in "agree" → "agreeing".

Other Spelling Changes

Some words undergo small but important changes when combined with suffixes. These adjustments help maintain pronunciation or prevent awkward spelling combinations.

Tips for Mastery

Practice writing words with their suffix forms, create flashcards, and pay attention to stress in multi-syllable words. Understanding these patterns will make your spelling much stronger.
